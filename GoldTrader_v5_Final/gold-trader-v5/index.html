MT5 install guide — XAUUSD_GoldTrader_Pro.mq5
1
Download the .mq5 file from the ZIP provided in this chat
2
Open MetaTrader 5 → File → Open Data Folder → MQL5 → Indicators
Copy XAUUSD_GoldTrader_Pro.mq5 into this folder
3
Compile: Press F4 → MetaEditor opens → find the file → press F7 to compile. No errors should appear.
4
Add to chart: Open XAUUSD → Navigator panel → Indicators → GoldTrader Pro → drag onto chart
5
Settings: Min Confluence = 5, ATR Mult = 1.2, TP1 RR = 2.0, TP2 RR = 3.0, TP3 RR = 5.0
6
Alerts: Enable Alerts = true, Enable Push = true → download MT5 mobile app and log in same account
7
Read signals: ▲ green arrow = BUY · ▼ red arrow = SELL · dotted green = TP lines · solid red = Stop Loss · score label "BUY 6/7" appears on each signal
