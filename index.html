⚡ Fix CORS — Deploy Vercel Proxy (3 min)
This creates a tiny free server that calls Anthropic on your behalf — solving the CORS issue permanently. Free on Vercel forever.
1
Go to vercel.com → Sign Up with GitHub
2
Click Add New → Project → Import Git Repository
3
Create a new GitHub repo called gtp-proxy and upload the proxy files from the ZIP (the gold-trader-proxy folder)
4
In Vercel, before deploying, go to Environment Variables and add:
Name: ANTHROPIC_API_KEY
Value: sk-ant-api03-your-key-here
5
Click Deploy. Vercel gives you a URL like:
https://gtp-proxy.vercel.app
6
Your proxy URL is:
https://gtp-proxy.vercel.app/api/claude
Paste this into the Proxy URL field on the left and click Save.
7
Click Refresh in the app — Claude AI now works with no CORS error ✓
